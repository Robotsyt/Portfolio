## Quick Start (local)
This repository is the static source for my portfolio site. To preview, open `index.html` in a browser; the site is intended to be hosted via GitHub Pages for deployment.

## Projects
Project pages are in the `projects/` folder — open any of the `.html` files to view project details and edit their content.

## Notes on Customization (personalized)
Brief summary: I updated this site to reflect my experience in systems administration, Power BI, workflow automation, Python, SQL, JavaScript, and Azure. Content lives in `index.html`, theme variables are in `assets/css/style.css`, project pages are in `projects/`, and images are stored in `assets/img/`. Prefer editing CSS variables for theme changes rather than inline styles.
